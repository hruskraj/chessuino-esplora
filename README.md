# Chessuino Esplora

Semestral project for BI-ARD. Chess on Arduino Esplora.

## Prerequisities

Arduino IDE

## Upload

Open main/main.ino in Arduino IDE. Connect Arduino Esplora. Upload the project into Arduino.
